# Execution Contracts

## Purpose

This folder defines governance constraints for executing actions with evidence, uncertainty, recovery, and human-review boundaries.

## Canonical Ownership

- Owner: Governance Recorder + Auditor
- Canonicality: Canonical execution governance contracts

## Belongs Here

- Evidence requirements before action.
- Bounded execution rules.
- Uncertainty surfacing rules.
- Runtime decision boundaries.
- Human-review thresholds.

## Does Not Belong Here

- Implementation code.
- Validation reports.
- Roadmap claims.
- Autonomous agent promises.
