# Review Gates

## Purpose

This folder defines when execution requires human review, safe automation boundaries, and validation escalation.

## Canonical Ownership

- Owner: Auditor + Governance Recorder
- Canonicality: Validation governance gate layer

## Belongs Here

- Human-review matrices.
- Safe automation boundary definitions.
- Validation escalation rules.

## Does Not Belong Here

- Raw test logs.
- Roadmap claims.
- Implementation code.
- Autonomous approval claims.

- Execution contracts: `01_GOVERNANCE/execution_contracts/`
- Execution maps: `03_OPERATIONS/execution_layer/`
